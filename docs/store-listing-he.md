## תיאור מלא (4000 תווים מקסימום)

יומן תזכורות הוא יומן ותזכורות עברי-RTL מלא, שעובד גם ללא חיבור לאינטרנט וללא הרשמה.
האפליקציה תוכננה במיוחד למשתמשים בישראל, עם גופן Heebo נעים, גודל טקסט מתכוונן, וצבעים נינוחים.

✨ הכל בעברית מלאה
✨ ללא פרסומות, ללא אנליטיקס, ללא מעקב
✨ עובד גם בלי אינטרנט
✨ סנכרון Google Drive אופציונלי בין הטלפון למחשב

### Data Safety form
**Data collected:**
- ☑ Files and docs → App database file in Drive appData (cloud sync, optional, encrypted in transit, user can delete via Google account)
- ☑ App activity → Local notification permission state (on-device only, never transmitted)

**Data shared:** None
**Data encrypted in transit:** Yes
**User can request data deletion:** Yes — via in-app "Sign out" or via Google account settings
